## Course

## Description
The Phonebook System is a console-based application developed in Python.
The system allows users to manage contacts efficiently with features such as
adding, updating, deleting, searching contacts, and organizing them into groups.

This project is developed as part of the Software Engineering course.

---

## Features
- Add, update, and delete contacts
- Search contacts by name, phone number, or email
- Mark contacts as favorites
- Create and manage contact groups
- Assign and remove contacts from groups
- Import and export contact data using text files
- Console-based user interface

---

## Technologies
- Programming Language: **Python**
- Interface: **Console / Terminal**
- Data storage: Text files (`.txt`)
